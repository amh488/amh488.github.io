Journey to the Maths

欢迎来到我的个人主页！这里记录了我在数学领域中的探索与思考。

学习目标

通过这个站点，我整理和分享我对各个数学分支的理解，比如微积分、线性代数、数论、几何等。

同时，我也会把自己在 GitHub 上做的一些小项目、实验性想法集中在这里，方便回顾与改进。

微积分笔记

目前主要在学习单变量和多变量微积分，关注函数、极限、连续、导数与积分等基础概念，并尝试用图形/直观的方式去理解。

核心概念

导数：描述函数在某一点附近的瞬时变化率，可以理解为“切线斜率”。

积分：描述某个量的累积效果，比如曲线下的面积，可以看成是“无穷小求和”的极限。

代数探索

代数部分主要包括线性代数和一些抽象代数的入门内容，比如向量空间、线性变换、矩阵、群论基础等。

线性代数要点

矩阵运算：包括矩阵乘法、逆矩阵、行列式等，是线性变换的代数表示。

向量空间：讨论“向量”的线性组合、基底、维度以及用矩阵表达的变换。

几何世界

从直观的欧几里得几何，到更抽象的非欧几何、拓扑学，我会记录理解过程中的一些关键想法和图形化理解。

几何思想

空间关系与对称性：通过几何变换（旋转、平移、反射等）来理解图形背后的不变性。

几何与物理：比如广义相对论用“弯曲的时空几何”来描述引力，本质上也是一种几何语言。
